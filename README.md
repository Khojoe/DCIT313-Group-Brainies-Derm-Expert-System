# DCIT313-Group-Brainies-Derm-Expert-System
DermExpert: A rule-based expert system for diagnosing common skin diseases like Acne, Eczema, Psoriasis, Ringworm, Dermatitis, and Scabies. Built with SWI-Prolog for inference and Python (pyswip) for the UI. This project by group Brainies simulates dermatologist reasoning with forward chaining and explanations. For educational purposes only.
